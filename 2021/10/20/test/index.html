hexo的部署

hexo的安装

在此之前请先安装node.js

安装hexo命令:

npm install -g hexo-cli

在建立的文件夹中输入命令：

hexo init
npm install

即可新建好文件夹

注：如果要在GitHub上搭建博客，在_config.yml的最后的delo后方输入你的仓库地址
以及blog的标题等信息也会在_config.yml中，请记得修改配置

有关更多配置参考官方文档 https://hexo.io/zh-cn/docs/configuration

在新建的文件中

hexo init

来新建网站

然后就可以使用

hexo new “title”

来新建文章了，文章将会存放在public文件夹中

由于hexo是静态网站，新建好文章后记得用

hexo g

生成页面

测试可用

hexo s

打开服务，并可以在游览器中输入localhost:4000中查看博客是否有问题

写好需要部署到GitHub或者服务器中

hexo d

若配置不对请检查一遍配置再进行上传

写在后面：
在利用github搭建hexo的过程中也遇到一些问题

例如：hexo d后GitHub中的自定义域名会被修改
解决：在hexo生成博客的source目录中新建CNAME文件，在文件中输入域名，以防custom domain被重置
